# PROJECT T&K — GitHub Pages Kurulumu

## Dosyaları kişiselleştirme

Aşağıdaki dosyaları kendi içeriklerinizle değiştirin. Dosya adlarını değiştirmeyin:

- `assets/photos/kapak.jpg` — ana düğün fotoğrafı
- `assets/photos/video-kapak.jpg` — videonun kapak görseli
- `assets/photos/01.jpg` ... `04.jpg` — anı fotoğrafları
- `assets/video/mesaj.mp4` — dikey veya yatay MP4 video

Daha fazla fotoğraf eklemek için `index.html` içindeki `<article class="memory">...</article>` bloğunu kopyalayın.

## GitHub üzerinden yayınlama

1. GitHub hesabında yeni bir repository oluşturun.
2. Repository adı olarak `tavsan-ve-kaplumbaga` yazabilirsiniz.
3. Bu klasördeki bütün dosyaları repository ana dizinine yükleyin.
4. Repository içinde `Settings > Pages` bölümünü açın.
5. `Build and deployment > Source` alanında `Deploy from a branch` seçin.
6. Branch olarak `main`, klasör olarak `/ (root)` seçip `Save` düğmesine basın.
7. Yayın adresiniz genellikle şu biçimde olur:
   `https://KULLANICI-ADINIZ.github.io/tavsan-ve-kaplumbaga/`

## Önemli

- GitHub Free kullanıyorsanız Pages için repository'yi `Public` yapmak en sorunsuz yöntemdir.
- Site adresi kesinleşmeden QR kodu afişe bastırmayın.
- Baskıdan önce QR kodu en az iki farklı telefonda ve mobil internetle test edin.
- Fotoğrafları mümkünse 1600–2200 piksel uzun kenar ve sıkıştırılmış JPG olarak kullanın.
- Video dosyasını ideal olarak 30–60 MB altında tutun.
